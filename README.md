# ⚔ Blade Idle MMORPG

An idle RPG built with HTML5 Canvas. Fight boss waves, recruit heroes, forge weapons, and ascend to Legend & Mystic tiers.

## 🎮 Play
👉 **[Play on GitHub Pages](https://porches2.github.io/Idle-Ascendent)**

## ✨ Features
- Pixel-art heroes with idle/walk/attack/death animations
- Day/night cycle with sun, moon, stars & clouds
- Champion → Legend → Mystic hero ascension tiers
- Animated fire crown & purple wing FX for elite heroes
- Weapon & armor forge system with rarity tiers
- Boss wave system with escalating difficulty
- **Google Sign-In** — save progress synced to the cloud via Firebase

## ☁ Cloud Save
Progress is saved automatically every 10 minutes to Firebase Firestore.
Sign in with Google to access your save from any device.

## 🛠 Tech
- Vanilla HTML5 Canvas (no frameworks)
- Firebase Auth (Google Sign-In)
- Firebase Firestore (cloud saves)
- Hosted on GitHub Pages
